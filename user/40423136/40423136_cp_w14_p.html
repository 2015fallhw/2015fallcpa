2015/12/18 利用brython繪製直線和垂直線

40423136 設計一甲 陳柏維

程式碼

# 導入 doc from browser import document as doc from browser import console import math # 準備繪圖畫布 canvas = doc["plotarea"] ctx = canvas.getContext("2d") # 開始畫直線 for i in range(11): ctx.beginPath() ctx.lineWidth = 5 ctx.moveTo(0,0+i*20 ) ctx.lineTo(200,0+i*20 ) ctx.strokeStyle = "#0000ff" ctx.stroke() for i in range(11): ctx.beginPath() ctx.lineWidth = 5 ctx.moveTo(0+i*20, 0) ctx.lineTo(0+i*20, 200) ctx.strokeStyle = "#FF0000" ctx.stroke()
