引號的使用規則

關於引號的使用,單引號跟雙引號能把裡面的字做輸出,但不能任意的切換這兩個引號,這點在輸出英文字串的時候尤其需要注意


例如: W1教的"print('___')"

W2上課內容:

使用Github倉儲(下載)
進入Github主頁→複製連結網址→開啟Ana3_lite2裡的start.bat
使用命令黑頁(cmd.exe)輸入檔案位置:
cd tmp→輸入git clone 貼上連結網址→輸入git checkout gh-pages 切換倉儲分枝→完成
使用Github倉儲(上傳)
輸入git add -A→git commit -m"上傳內容訊息"→git config user.email "信箱"→git config user.name "名字"
→git push origin gh-pages→完成
