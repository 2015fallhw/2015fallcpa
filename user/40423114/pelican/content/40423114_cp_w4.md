###LEO EDITOR TUTORIAL解說

Leo Editor是一個功能特殊的軟體套件, 不 
僅能作為集思廣益用的大綱編輯, 程式編輯 
架構器, 電子書編輯器, 也是絕佳的網站管理 
套件。

Leo Editor雖然採用 Python 程式語言開發, 許 
多建構理念也都圍繞著 Python 物件開發的特 
性打造, 但是編輯管理的內容卻不限於Python 
相關的程式, 而可及於各式複雜的結構化資料 。
