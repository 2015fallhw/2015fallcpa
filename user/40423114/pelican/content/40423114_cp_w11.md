###修正fix_leo-editor安裝### 

開啟教學網頁

點選2015fall課程

下載fix

將檔案解壓縮

搜尋 leo editor github

進入網頁

點選leo-editor

點選release

點選broke-abbrev

點選Source code(zip)

將檔案移至fix資料夾

到Ana3_lite2\data\SciTE

開啟SciTe.exe

使用此編輯器開啟fix.bat

在最後增加一行指令 指令為 copy leo-editor-broke-abbrev.zip C:
\Ana3_2015\data\

儲存檔案

使用此編輯器開啟start.bat

將原指令 (第35行) pip install http://140.130.17.17/public/leo-editor-master.zip 

更改為 pip install leo-editor-broke-abbrev.zip

儲存檔案

點選開啟fix.bat

點選開啟start.bat
