Python是甚麼?

Python 是一套功能強大、直譯式並且物件導向的程式語言

Python 的特點:
1.容易撰寫
2.功能強大
3.跨平台
4.容易擴充

Python的應用


1.Zope 應用程式伺服器與相關系統
2.科學計算
3.網路應用
4.圖形介面程式
5.作業系統管理

一些Python指令1

sys模組：
argv: 命令列參數。
exit([arg]): 結束程式。
exitfunc: 若有指定此函式，程式結束前會先呼叫此函式。

一些Python指令2

os模組：
chdir(path): 變更目前工作目錄。
getcwd(): 取得目前工作目錄。
getenv(varname[, defaultValue]): 取得環境變數。
putenv(varname, value): 設定或新增環境變數。
listdir(path): 傳回指定路徑的內容。
remove(path): 刪除指定的檔案。
removedirs(path): 以遞迴的方式刪除指定的路徑。
rename(src, dst): 變更路徑或檔案名稱。
