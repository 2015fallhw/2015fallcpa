PYTHON

Python 與 shell scripts 相似，是直譯式程式。互動式程式，執行過程中可與程式互動。跨平台，未經修改下可在Linux，Windows，MacOS X 和其他平台上使用。易讀、易學、易維護，適合初學者。可設計各種應用程式，從文字處理、www 網頁瀏覽器、管理倉儲都可完成

python控制結構

for 為 Python 的關鍵字之一，後面緊接一個取得元素的變數，此例中用 i ，然後接另一個關鍵字 in ，其後就是複合資料型態的資料，最後接冒號，底下縮排就是迴圈所要重複的工作。這一行用中文來解釋，就是從 a 中取出元素放到 i 裡，迴圈直接取得複合資料型態的元素，若是沒有元素，迴圈自動結束。
