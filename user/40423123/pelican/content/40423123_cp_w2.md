建立與刪除倉儲
=============
                                              
                                              
                                                                         
建立倉儲
--------------
                            
1. 點選右上角的"+"
2. 點選"NEW REPOSITORY"
3. 輸入欲建立之倉儲名稱
4. 勾選"INITIALIZE THIS REPOSITORY WITH A READ ME"
5. 點選 "CREATE REPOSITORY"

刪除倉儲
-------------
                                               
1. 點選你要刪除的倉儲
2. 點選右邊的"SETTING"
3. 往下滑
4. 點選最後一個"DELETE THIS REPOSITORY"
5. 輸入你要刪除的倉儲名稱
6. 按下"I UNDERSTAND THE CONSEQUENCES, DELETE THIS REPOSITORY"
