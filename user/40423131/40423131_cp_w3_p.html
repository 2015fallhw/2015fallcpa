課程內容

Github


GitHub 同時提供付費帳戶和免費帳戶。
這兩種帳戶都可以建立公開的倉庫，
但是付費享有建立私人的倉庫。

Github指令

........超多

git clone :複製一個專案

git branch :列出所有近端分支

git branch (分支名稱) :建立近端分支

git checkout (分支名稱) :切換近端分支

git add :將現有檔案暫存=(拍快照)

git commit :將暫存檔案建立在一個*節點。

(節點:就是未來你可以回朔及追蹤的參考)

git pull (遠端名稱) (分支名稱) :下載一個遠端分支並合併於近端

git push :推送分支
