LEO?

Leo Editor 就是採 Python 的程式方法與流程來控管其中的各節點與資料內容，

leo是一種編輯器，可以編輯網頁、製作投影片也可以當成是一個交流程式的工具

使用LEO編輯投影片和網頁時，記得要注意版本，如果是使用5.0版本，可能在5.1版本上無法開啟使用或檔案出問題。

LEO語言

我們可以透過老師的reveal.leo檔，製作我們的投影片和作業，可以透過leo編輯器，在個人網頁中加入影片或圖片等資料。

leo的語言就是一定要記得有一個開始就要有一個結束。

可以利用html語法套用在LEO編輯器，編輯我們的資料。

例如:

    *字代表小於大於符號

  • 只要在*加入"ul"的碼，便可製出有小圓頭的清單 ，接下來在每個清單前面都要在*加入"li"，每個清單都要在li前面加入/已表示結束，然後再資料輸入後在*加入"/ul"。
  • 在*內加入h1，打入標題文字，再以*字內加入/h1。
  • 每個語法都以相同方式做開始與結束，就能完成投影片編輯。

Jupyter

上課時，老師都用Jupyter使用python3。

Jupyter 是把IPython和Python解釋器剝離後的產物，將逐漸替代IPython 獨立發行。Jupyter 可以和Python 之外的程序結合，提供新的、強大的服務。例如:Ruby REPL環境IRuby 和Julia REPL環境IJulia。

我們可以利用Jupyter去學習Python語言，讓自季更熟悉Python。
