Git 指令教學

  • 下載剛剛建立好的倉儲 ，指令為 ：
     git clone 倉儲網址
  • 需建立一個網站的分支 ，指令為 ：
     git branch 分支名稱
  • 必須查詢目前所在分支 ，指令為 ：
     git branch
  • 切換我想要的指定分支 ，指令為 ：
     git checkout 分支名稱
